# AGENTS.md

This repository provides the **java-unit-testing** Agent Skill.

Before writing, reviewing, or refactoring Java tests, read `SKILL.md` at the
repository root and apply its rules. Detailed patterns are in the `references/`
directory:

- `references/PATTERNS.md`: code for every test pattern.
- `references/ASSERTJ.md`: AssertJ usage and correctness traps.
- `references/UNIT-TESTING-PRACTICES.md`: state-first testing, test doubles,
  determinism, coverage.
- `references/CHECKLIST.md`: pre-flight checklist and pitfalls.

`SKILL.md` is the single source of truth. This file adds no rules of its own; it
exists so agents that discover instructions through `AGENTS.md` are routed to
the skill.
